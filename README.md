# Proyecto WordPress con Docker - DAM

Este proyecto consiste en la creación de un blog en WordPress utilizando Docker y Docker Compose,
como parte de la actividad de la Unidad 3 del ciclo de Desarrollo de Aplicaciones Multiplataforma (DAM).

El blog incluye varias páginas y entradas donde se tratan temas como la gestión de archivos en
Windows y Linux, sistemas de particionado y almacenamiento RAID.

## Tecnologías utilizadas
- Docker
- Docker Compose
- WordPress
- MySQL
- phpMyAdmin

## Puesta en marcha
Para iniciar el proyecto es necesario ejecutar el siguiente comando desde la carpeta del proyecto:

docker-compose up -d

Una vez levantados los contenedores, se puede acceder a WordPress desde el navegador usando
la IP del equipo y el puerto configurado.
